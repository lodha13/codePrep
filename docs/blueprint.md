# **App Name**: CodePrep Pro

## Core Features:

- Quiz Selection: Allow users to select quizzes based on skill or assigned tests.
- Multiple Choice Questions: Present multiple choice questions to the user.
- Code Editor with Intellisense: Provide a code editor with intellisense for coding questions in various languages, including Java.
- Code Execution and Testing: Allow users to execute code and test against predefined test cases.
- Test Result Capture: Capture and store test results for each user and quiz.
- Admin Panel: Provide an admin panel to upload questions in JSON format and assign tests to users.
- AI Enhanced Evaluation Tool: Use Gemini AI to act as a tool to automatically grade coding submissions, based on correctness and code quality. Note: the final grading decisions rests with the user/admin, but AI may assist with comments/suggestions about the submission.
- Email Result Delivery: Allow admin to send test results to users via email.
- Question Storage: Store quiz questions and related metadata in Firestore.

## Style Guidelines:

- Primary color: Deep blue (#1A535C) for a professional and reliable feel.
- Background color: Light blue (#E8F6F8) for a clean and calm interface.
- Accent color: Orange (#FFA630) for highlights and call-to-action buttons.
- Body font: 'Inter', sans-serif, for clean and readable text.
- Headline font: 'Space Grotesk', sans-serif, for a tech-forward feel. Used in combination with Inter, for longer text.
- Code Font: 'Source Code Pro' for the display of computer code.
- Use clear and concise icons to represent different quiz categories and actions.
- Clean and intuitive layout for easy navigation and readability.
- Subtle animations for feedback on actions like submitting answers or running code.
